# Epic 1: MVP Piškvorky

## Epic ID
1

## Title
MVP Piškvorky - Základní implementace hry

## Description
Implementace základní funkčnosti hry piškvorky 15×15:
- Interaktivní mřížka 15×15
- Střídání tahů mezi hráči (X/O)
- Detekce výhry (5 v řadě)
- Detekce remízy
- Reset hry
- Responzivní UI

## Status
Draft

## Acceptance Criteria

1. Hra je hratelná end-to-end (nový tah → výhra/remíza/reset)
2. Správná detekce všech výherních kombinací
3. UI pracuje na mobilu i desktopu (360-1440px)
4. Všechny tahy se správně zaznamenávají

## Stories

### 1.1: Vykreslit mřížku a střídání tahů
### 1.2: Detekce výhry a remízy
### 1.3: Reset hry a UI vylepšení

## Dependencies
Žádné

## Notes
Fokus na MVP - žádný AI, multiplayer ani persistence.
