## Description
Přidání diskrétního footeru na spodek stránky s:
- Copyright textem © polyweb.cz
- Textem o BMAD framework testu
- Odkazy na: polyweb.cz, BMAD dokumentaci a GitHub repozitář (vše v novém okně)
- Design: minimalistický, nenápadný, bez rušení hry

Footer bude nedílnou součástí stránky, bez zbytečné viditelnosti.

- FR1: Zobrazení copyright informací
- FR2: Kredity pro BMAD framework
- FR3: Relevantní odkazy (polyweb.cz, BMAD, GitHub repo)
- FR4: Diskrétní design (nenápadný)

## Technical Notes

### HTML Struktura

```html
<footer class="footer">
  <p>
    © 2025 <a href="https://polyweb.cz" target="_blank" rel="noopener noreferrer">polyweb.cz</a>
    —
    Vytvořeno k testu <a href="https://docs.claude.com" target="_blank" rel="noopener noreferrer">BMAD framework</a>
    —
    <a href="https://github.com/polyweb-cz/pisvorky-js" target="_blank" rel="noopener noreferrer">GitHub</a>
  </p>
</footer>
```

### CSS Stylování

```css
.footer {
  margin-top: 2rem;
  padding: 1.5rem 1rem;
  text-align: center;
  border-top: 1px solid #e0e0e0;
  background-color: #fafafa;
}

.footer p {
  margin: 0;
  font-size: 0.85rem;
  color: #666;
  line-height: 1.6;
}

.footer a {
  color: #0066cc;
  text-decoration: none;
  transition: color 0.2s ease;
}

.footer a:hover {
  color: #004499;
  text-decoration: underline;
}

@media (max-width: 768px) {
  .footer {
    padding: 1rem 0.75rem;
    font-size: 0.8rem;
  }
}
```

### Bezpečnost

- `target="_blank"` - Otevírá v novém okně
- `rel="noopener noreferrer"` - Prevence XSS útoků (nový window nemá přístup k window.opener)
