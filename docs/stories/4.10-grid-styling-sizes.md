# Story 4.10: Grid Styling for Sizes

## Epic
Epic 4: Design & UI Refresh

## User Story

Jako uživatel
Chci, aby se herní plocha dynamicky měnila dle vybrané velikosti
Abych měl odpovídající vizuální zážitek na 3×3, 10×10 i 15×15

## Description

Implementace CSS pro dynamické přizpůsobení herní mřížky třem velikostem. Každá velikost bude mít:
- Správné `grid-template-columns` (3 vs 10 vs 15 sloupců)
- Proporcionální font-size pro buňky
- Responsive design zachován

## Acceptance Criteria

1. **AC1:** CSS grid se změní dle vybrané velikosti (3/10/15 sloupců)
2. **AC2:** Buňky jsou proporcionální (zmenšují se na menších gridech)
3. **AC3:** Grid max-width: 600px na všech velikostech
4. **AC4:** Responsive design zachován (360px, 768px, 1440px)
5. **AC5:** Font-size se přizpůsobí velikosti buněk

## Files Changed

### Modified Files
- `src/styles.css` - CSS třídy `.size-3`, `.size-10`, `.size-15` pro grid

## Technical Notes

### CSS Grid Classes
```css
.grid-container.size-3 {
    grid-template-columns: repeat(3, 1fr);
}

.grid-container.size-10 {
    grid-template-columns: repeat(10, 1fr);
}

.grid-container.size-15 {
    grid-template-columns: repeat(15, 1fr);
}
```

### Font-size Scaling
Buňky na 3×3 mohou mít větší font (více prostoru), na 15×15 menší (více políček).
