# Story 4.14: Hamburger Button & Toggle Logic

## Epic
Epic 4: Design & UI Refresh

## User Story

Jako uživatel
Chci mít hamburger menu pro zobrazení/skrytí nastavení
Abych měl čistší rozhraní a nastavení na vyžádání

## Description

Přidání hamburger buttonu (☰ emoji) vedle indikátoru "NA TAHU" pro toggle viditelnosti nastavení. Ikona se změní na × když je menu otevřené. Smooth animace pomocí CSS transitions.

## Acceptance Criteria

1. **AC1:** Hamburger button (☰ emoji) je vedle "NA TAHU" indikátoru v `.game-header`
2. **AC2:** Button má styling: padding, border, border-radius, hover efekt, cursor pointer
3. **AC3:** Kliknutí togglene viditelnost `.controls-fieldset` (přidá/odebere třídu `.open`)
4. **AC4:** Button má ARIA attributes (`aria-label="Nastavení"`, `aria-expanded`)
5. **AC5:** Ikona se mění: ☰ (zavřeno) → × (otevřeno)
6. **AC6:** Smooth animace slide down/up pomocí `max-height` transition
7. **AC7:** Všechny testy procházejí (včetně 8 nových)

### CSS Styling

```css
    outline: none;
    border-color: #EB5E00;
    box-shadow: 0 0 3px rgba(235, 94, 0, 0.3);

```

### JavaScript Toggle Method
```javascript
toggleSettings() {
    if (!this.hamburgerButton || !this.controlsFieldset) return;

    const isOpen = this.controlsFieldset.classList.toggle('open');
    localStorage.setItem('settingsMenuOpen', isOpen ? 'true' : 'false');

    this.hamburgerButton.textContent = isOpen ? '×' : '☰';
    this.hamburgerButton.setAttribute('aria-expanded', isOpen ? 'true' : 'false');
}
```
