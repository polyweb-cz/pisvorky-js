## Technical Notes

### Current HTML Structure
```html
<header class="game-header">
    <div class="turn-indicator">...</div>
    <div class="controls">...</div>
</header>
```

HTML struktura je OK, elementy budou zalamovány flexboxem.

### CSS Changes
- `.game-header`: Změnit z `text-align: center` na flexbox
  - `display: flex`
  - `align-items: center`
  - `justify-content: center`
  - `gap: 2rem`
  - `flex-wrap: wrap`
  - `margin-bottom: 1rem`

- `.turn-indicator`: Odebrat `margin-bottom: 1rem`

- `.controls`: Odebrat `margin-top: 1rem`

### Why Flexbox?
- Jednoduchý a deklarativní
- Automatické zarovnání a spacing
- Vestavěná responsivita s `flex-wrap: wrap`
- Kompatibilní se všemi moderními prohlížeči

## Risk Assessment

**Risk 1:** Změna layoutu v headeru mohla by zatvrdit jiné komponenty
- **Mitigation:** Flexbox na `.game-header` neovlivní `.game-main` (mřížka) - jsou v různých containerech

**Risk 2:** Na velmi malých displejích (320px) se prvky nemusí vejít
- **Mitigation:** `flex-wrap: wrap` zajistí zalomení, pokud by to bylo potřeba
